#Marionette Forms

This application uses **grunt** as task manager and some dependences like **jquery**, **backbone**, **marionette**, etc.

    └── readme.md

## Installation

Run npm install ([**node**](http://nodejs.org) and [**npm**](https://npmjs.org) previously installed)

    $ npm install
